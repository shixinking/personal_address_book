# personal_address_book

一个基于 Java 的简易个人通讯簿管理系统，支持控制台交互，可实现联系人的增删改查、分类管理与数据持久化。

---

## 📌 项目简介
本项目实现了一个基础的通讯录管理系统，满足以下核心需求：
- 支持按姓名、地区、电话等属性查询和浏览联系人
- 可对联系人信息进行添加、删除、修改操作
- 支持联系人分类管理（如家人、朋友、同事等）
- 数据支持本地文件持久化存储，重启程序后数据不丢失
- 内存中使用 Java 集合类管理联系人数据，当前版本采用控制台交互，后续可扩展 GUI 界面与数据库支持

---

## ✨ 功能特性
- **联系人管理**：新增、删除、修改、查询联系人信息
- **分类管理**：支持自定义联系人分组，按类别筛选联系人
- **数据持久化**：使用本地文件保存数据，程序关闭后自动保存，启动时自动加载
- **控制台交互**：纯命令行操作，简单易用，适合作为 Java 基础练习项目

---

## 🛠️ 技术栈
- 编程语言：Java 11
- 开发工具：IDEA / PhpStorm
- 核心技术：Java 集合框架、文件 I/O、控制台交互
- 数据存储：本地文本文件（后续可扩展为 MySQL 数据库）
